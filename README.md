# Student Performance Prediction

## Objective

Predict student performance based on various academic and demographic factors.

## Dataset

Student dataset containing features like study time, parental education, and scores.

## Tools Used

* Python (Pandas, NumPy)
* Visualization (Matplotlib, Seaborn)
* Machine Learning (Scikit-learn)

## Analysis Performed

* Data preprocessing and encoding
* Exploratory data analysis
* Feature relationship analysis

## Models Used

* Linear Regression
* Decision Tree
* Random Forest

## Results

* Machine learning models successfully predicted student scores
* Identified key factors affecting performance

## Key Insights

* Study time strongly impacts performance
* Parental background influences results
* Some factors have higher predictive importance

## Future Work

* Improve model accuracy with advanced techniques
